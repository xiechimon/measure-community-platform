数智化社区服务平台 API (v1)

Download OpenAPI specification:

六个业务模块的聚合 API 契约快照，供 Apifox/Postman 导入；所有外部请求经网关(9090)访问。

模块对接状态

模块 状态 说明
🟢 鉴权 (auth) ✅ 可对接 登录、用户、角色(CRUD+权限分配)、组织树 全部就绪
🟢 人口信息 (info) ✅ 可对接 人口档案分页查询、录入、版本更新、变更历史 全部就绪
🟡 首页门户 (portal) ⏳ 占位 仅 /api/v1/portal/ping 健康探针，业务接口待开发
🟡 社区服务 (service) ⏳ 占位 仅 /api/v1/service/ping 健康探针，业务接口待开发
🟡 社区公益 (welfare) ⏳ 占位 仅 /api/v1/welfare/ping 健康探针，业务接口待开发
🟡 居务管理 (affairs) ⏳ 占位 仅 /api/v1/affairs/ping 健康探针，业务接口待开发

对接须知

  • 网关地址: http://localhost:9090（生产按实际域名替换）
  • 鉴权方式: Bearer JWT，先调 /api/v1/auth/login 获取 token，后续请求头带 Authorization: Bearer <token>
  • 统一响应格式: { "code": 200, "message": "...", "data": ... }
  • 分页请求统一用 page(页码,从1开始) + size(页大小) 参数
  • 分页响应统一含 records(当前页数据) + total(总条数) + size + current + pages
